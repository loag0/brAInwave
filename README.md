# brAInwave

brAInwave is a study planner that leverages AI and machine learning to suggest optimal study times. Its goal is to help users overcome procrastination and improve time management.

## Features

- AI-driven study time recommendations
- Personalized scheduling
- Tools to track progress and habits

## Getting Started

1. Install dependencies
2. Configure your study preferences
3. Start planning with brAInwave
